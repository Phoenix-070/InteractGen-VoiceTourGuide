# Icon Setup

The extension requires three icon files in the `icons/` folder:
- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)  
- `icon128.png` (128x128 pixels)

## Quick Setup

### Option 1: Use the Icon Generator

1. Open `create-icons.html` in your browser
2. Click "Generate Icons"
3. Click "Download" for each size
4. Save them in the `icons/` folder

### Option 2: Create Simple Icons

You can use any image editor or online tool to create simple microphone icons:
- Use a microphone emoji (ðŸŽ¤) or icon
- Use a gradient background (purple/blue)

### Option 3: Use Placeholder Icons

For testing, you can create simple colored squares:
- 16x16, 48x48, 128x128 PNG files
- Any color (purple/blue recommended)
- Save as icon16.png, icon48.png, icon128.png

**Note**: The extension will work without icons, but Chrome will show a default icon. Icons are recommended for a polished look.
